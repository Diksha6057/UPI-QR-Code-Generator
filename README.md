# UPI QR Code Generator

A simple Python project that generates UPI QR Codes that can be scanned by any UPI application such as Google Pay, PhonePe, or Paytm.

## Features

- Generate UPI payment QR codes
- Accept custom UPI IDs
- Save QR codes as PNG images
- Simple and beginner-friendly implementation

## Complete Flow

User enters UPI ID
        ↓
Create UPI payment URL
        ↓
Generate QR code
        ↓
Save QR image
        ↓
Display QR image

## Technologies Used

- Python
- qrcode library
- Pillow
